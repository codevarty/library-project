# library-project
도서 대출 및 신청 프로젝트

## 목표
도서관처럼 현재 관리 중인 도서를 대출하고 보유중이지 않은 도서를 신청하는 프로젝트

## 개발 환경
- Spring Boot 3.3
- JAVA 17
- JWT
- Redis
- MySQL 8

## 작업 내용
- [ ] 회원 인증 (JWT) 기능 추가하기
- [ ] 도서 리스트 불러오기 및 검색 기능 추가
- [ ] 도서 등록, 수정, 삭제 (CRUD) 구현
- [ ] 사용자 도서 대출 및 반납 기능 추가
- [ ] 도서 대출 내역 기능 추가
- [ ] Naver search API를 사용해서 신청할 도서 검색 기능 추가
- [ ] 도서 신청 기능 추가
- [ ] 도서 신청 내역 기능 추가
- [ ] 관리자 도서 신청 승인 및 도서 거부 기능 추가
